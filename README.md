# LikeDAO

## Prerequisite

- asdf https://asdf-vm.com/#/core-manage-asdf?id=install
- Docker Compose https://docs.docker.com/compose/install/

## Requirements

- Golang 1.18
- Node 16
- Yarn 1.22.17

Managed by [asdf](https://github.com/asdf-vm/asdf)

## Quick Start

```
make setup
docker-compose up

# In another tab

make -C react-app dev
```

Then visit http://localhost:3000
